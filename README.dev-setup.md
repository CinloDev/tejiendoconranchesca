Instalación y desarrollo (setup inicial)

1. Instalar dependencias

```bash
npm install
```

2. Ejecutar en desarrollo

```bash
npm run dev
```

Notas:
- Este scaffold usa Next.js 14+ con App Router y MDX (via @next/mdx). Revisa `package.json` y ajusta versiones locales.
- Para Netlify se incluye `netlify.toml` y `_redirects` como pauta; Netlify puede requerir configuración adicional para aplicaciones Next.
